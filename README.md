# Global-Mental-Health-Analysis-2025-
Analysis of Global Mental Health trends using data analytics tools

Project Overview

This project explores global mental health trends using data analysis techniques. The goal is to identify patterns, risk factors, and regional differences in mental health outcomes.

Dataset

Global mental health dataset
Includes variables such as:

• Country
•	Mental health indicators
•	Demographics

Tools Used

	•	Excel / Power Query (Data Cleaning)
  
	•	Power BI (Data Visualization)

  Analysis Performed
  
	•	Trends in mental health indicators over time
  
	•	Comparison across countries and regions
  
	•	Identification of high-risk groups
